 Weather App
A simple, interactive weather application built with JavaScript, HTML, and CSS.
It fetches real-time weather data using the [OpenWeather API](https://openweathermapipt async/macro/microtask logging in a user-friendly console panel.
---
 Features
•  Search weather by city name
•  Displays:
	◦ City & Country
	◦ Temperature (°C)
	◦ Weather condition
	◦ Humidity
	◦ Wind speed
•  Logs sync, async, microtask, and macrotask events in a styled console output
•  Maintains a history of the last 5 searched cities (stored in localStorage)
•  Clean, responsive UI with styled chips for quick re-search
